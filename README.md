### Installasi In TERMUX

```bash
> apt update && apt upgrade -y
> pkg install git -y
> pkg install bash -y
> git clone https://github.com/MrG3P5/SELF-BOT.git
> cd SELF-BOT
> bash install.sh
> npm start
```
